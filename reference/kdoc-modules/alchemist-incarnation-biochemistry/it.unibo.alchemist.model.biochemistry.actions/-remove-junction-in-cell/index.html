RemoveJunctionInCell

class RemoveJunctionInCell : AbstractNeighborAction<T> (source)

Represent the action of removing a junction between the current node and a neighbor. This action only removes the junction reference inside this node; the neighbor totally ignores that a junction has been removed. This is a part of the junction removal process. See RemoveJunctionInNeighbor for the other part of the process
